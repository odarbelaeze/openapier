# openapier

A openapi v3 spec generator for go.

Heavily inspired by [swaggo/swag][swag].

## Supported Tags

You can find a list of all the supported `@` tags for both spec-level and operation-level annotations in the [TAGS.md](./TAGS.md) file.

To regenerate this documentation, run:

```bash
make docs
```

[swag]: https://github.com/swaggo/swag
